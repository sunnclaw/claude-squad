## Research Criteria

**Required:**
- Owner-operated (not corporate chains like Aspen Dental, Western Dental)
- 1-5 locations maximum
- Active phone number
- Direct email address (not contact form only)

**Preferred:**
- Google Business listing with reviews
- Practice size: 1-3 dentists
- Services include general/family dentistry
- Active website

---

## Research Sources

### Primary Sources
1. **Google Maps** - Search "dentist [city]" and filter by:
   - Locally-owned appearance
   - Reviews < 500 (avoid chains)
   - Owner name visible

2. **NPI Registry** - https://npiregistry.cms.hhs.gov/
   - Search by city and specialty
   - Filter by organization type

3. **Yelp** - Filter by "locally owned" and review count

4. **State Dental Board License Lookup**
   - Texas: https://tsbde.texas.gov/
   - Colorado: https://dpo.colorado.gov/Dentistry

### Data Points to Collect

For each clinic, collect:
1. Practice name
2. NPI number (if available)
3. Address
4. City
5. State
6. Timezone
7. Phone number
8. Owner first name
9. Owner last name
10. Owner title
11. Primary contact name
12. Primary contact role
13. Clinic size proxy
14. Website
15. Email address
16. Notes

4. Verify phone number is active

### Step 3: Format for Tracker (15 min)
