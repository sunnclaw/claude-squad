# Roadmap Scorecard

Score each item from 1-5.

## Inputs

- Revenue impact:
- Confidence:
- Time-to-value:
- Strategic importance now:
- Delivery effort:
- Ongoing maintenance:

## Formula

Use:

`Priority Score = (Revenue impact + Confidence + Time-to-value + Strategic importance) - (Delivery effort + Ongoing maintenance)`

## Rules

- Anything below `6` does not enter the active roadmap by default.
- If confidence is `1` or `2`, turn it into an experiment, not a full project.
- If delivery effort is `5`, break the project into a smaller shippable slice first.
- Never run more than 2 high-effort items at the same time.

## Decision

- Commit now
- Queue later
- Convert to experiment
- Reject
