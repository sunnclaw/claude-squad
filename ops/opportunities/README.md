# Opportunities

Use this folder for revenue-linked work only.

Every file here should describe a real opportunity, active customer need, or constrained experiment tied to near-term business value.

Recommended naming:

`YYYY-MM-DD-short-name.md`

Minimum contents:

1. Customer or segment
2. Problem
3. Revenue impact
4. Needed technical move
5. Smallest viable delivery
6. Kill condition
