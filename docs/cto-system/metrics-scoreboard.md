# Metrics Scoreboard

Review weekly.

## Revenue Metrics

- New leads
- Qualified leads
- Trials started
- New paid accounts
- Monthly recurring revenue
- Average revenue per account
- Churn rate

## Funnel Metrics

- Visitor to lead conversion
- Lead to demo conversion
- Demo to paid conversion
- Sign-up completion rate
- Activation rate
- Time to first value

## Delivery Metrics

- Deploys per week
- Lead time from commit to production
- Production incidents
- Change failure rate
- Mean time to recovery

## Efficiency Metrics

- Support tickets per active customer
- Manual ops hours per week
- Infrastructure cost as percent of revenue
- Engineering time spent on roadmap vs reactive work

## Rules

- Each metric needs one owner.
- Each metric needs a weekly target or threshold.
- If a metric is red for 2 consecutive weeks, it becomes a candidate for the next sprint.
- Remove any metric that does not drive a decision.
