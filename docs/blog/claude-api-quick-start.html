Anthropic的Claude API是构建AI应用的重要工具。本文将带你从零开始，掌握Claude API的核心用法，包括认证、请求构建、流式响应、错误处理等关键技能。

前置要求：
  • 基本的编程知识（Python或JavaScript）
  • Anthropic API密钥（从console.anthropic.com获取）
  • 了解REST API的基本概念

一、快速开始

1.1 获取API密钥

首先，你需要在Anthropic控制台创建API密钥：

  1. 访问 console.anthropic.com
  2. 登录或注册账户
  3. 进入API Keys页面
  4. 点击"Create Key"生成新密钥
安全提示：API密钥相当于账户密码，切勿泄露或提交到代码仓库。建议使用环境变量存储。

1.2 安装SDK

Anthropic提供官方SDK，支持Python和TypeScript：

Python:

pip install anthropic

TypeScript/JavaScript:

npm install @anthropic-ai/sdk

1.3 第一个API调用

让我们发送第一个请求给Claude：

Python示例：

import anthropic

client = anthropic.Anthropic(
    api_key="your-api-key-here"
)

message = client.messages.create(
    model="claude-sonnet-4-6-20250514",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "你好，请介绍一下你自己"}
    ]
)

print(message.content)

TypeScript示例：

import Anthropic from '@anthropic-ai/sdk';

const client = new Anthropic({
  apiKey: 'your-api-key-here',
});

const message = await client.messages.create({
  model: 'claude-sonnet-4-6-20250514',
  max_tokens: 1024,
  messages: [
    { role: 'user', content: '你好，请介绍一下你自己' }
  ]
});

console.log(message.content);

二、核心概念

2.1 模型选择

Anthropic提供多个Claude模型，适用于不同场景：

模型 特点 适用场景
Claude Opus 4.6 最强推理能力 复杂分析、研究任务
Claude Sonnet 4.6 平衡性能与成本 日常开发、对话应用
Claude Haiku 4.5 快速响应 实时对话、批量处理

2.2 消息格式

Claude API使用消息列表格式，支持多轮对话：

messages = [
    {"role": "user", "content": "什么是机器学习？"},
    {"role": "assistant", "content": "机器学习是人工智能的一个分支..."},
    {"role": "user", "content": "能举个例子吗？"}
]

2.3 System提示

System提示用于定义Claude的角色和行为边界：

message = client.messages.create(
    model="claude-sonnet-4-6-20250514",
    max_tokens=1024,
    system="你是一位专业的Python开发教练，用简洁易懂的方式回答问题。",
    messages=[
        {"role": "user", "content": "如何学习Python？"}
    ]
)
最佳实践：System提示应该清晰、具体，避免模糊的指令。好的System提示能显著提升输出质量。

三、进阶功能

3.1 流式响应

对于长文本生成，流式响应可以提供更好的用户体验：

with client.messages.stream(
    model="claude-sonnet-4-6-20250514",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": "写一篇关于AI的短文"}
    ]
) as stream:
    for text in stream.text_stream:
        print(text, end="", flush=True)

3.2 多模态输入

Claude支持图像输入，可以分析图片内容：

import base64

# 读取图片
with open("image.png", "rb") as f:
    image_data = base64.standard_b64encode(f.read()).decode("utf-8")

message = client.messages.create(
    model="claude-sonnet-4-6-20250514",
    max_tokens=1024,
    messages=[
        {
            "role": "user",
            "content": [
                {
                    "type": "image",
                    "source": {
                        "type": "base64",
                        "media_type": "image/png",
                        "data": image_data
                    }
                },
                {
                    "type": "text",
                    "text": "请描述这张图片的内容"
                }
            ]
        }
    ]
)

3.3 工具使用（Function Calling）

让Claude能够调用外部工具和API：

tools = [
    {
        "name": "get_weather",
        "description": "获取指定城市的天气信息",
        "input_schema": {
            "type": "object",
            "properties": {
                "city": {
                    "type": "string",
                    "description": "城市名称"
                }
            },
            "required": ["city"]
        }
    }
]

message = client.messages.create(
    model="claude-sonnet-4-6-20250514",
    max_tokens=1024,
    tools=tools,
    messages=[
        {"role": "user", "content": "北京今天天气怎么样？"}
    ]
)

# Claude可能会返回tool_use块
if message.stop_reason == "tool_use":
    for block in message.content:
        if block.type == "tool_use":
            print(f"工具: {block.name}")
            print(f"参数: {block.input}")

四、错误处理

4.1 常见错误类型

错误代码 含义 解决方案
401 认证失败 检查API密钥是否正确
429 请求过于频繁 实现重试机制，使用指数退避
500 服务器错误 稍后重试
prompt_too_long 输入超过限制 减少输入长度或切换模型

4.2 实现重试机制

import time
from anthropic import APIError, RateLimitError

def call_with_retry(client, messages, max_retries=3):
    for attempt in range(max_retries):
        try:
            return client.messages.create(
                model="claude-sonnet-4-6-20250514",
                max_tokens=1024,
                messages=messages
            )
        except RateLimitError:
            wait_time = (2 ** attempt) + 1
            print(f"遇到限流，等待{wait_time}秒后重试...")
            time.sleep(wait_time)
        except APIError as e:
            if attempt == max_retries - 1:
                raise
            time.sleep(1)
    return None

五、最佳实践

5.1 Prompt设计原则

  • 清晰具体：明确告诉Claude你想要什么
  • 提供上下文：给出必要的背景信息
  • 示例驱动：复杂任务提供输出示例
  • 分步骤：复杂任务拆解为多步

5.2 成本优化

节省API成本的技巧：
  • 选择合适的模型：简单任务用Haiku
  • 控制max_tokens：避免不必要的长输出
  • 缓存常用上下文：减少重复发送
  • 批量处理：合并相似请求

5.3 安全考虑

  • 不要在客户端暴露API密钥
  • 验证和清理用户输入
  • 设置合理的速率限制
  • 记录和监控API使用情况

六、完整示例：构建简单的AI助手

import anthropic
import os

class ClaudeAssistant:
    def __init__(self):
        self.client = anthropic.Anthropic(
            api_key=os.environ.get("ANTHROPIC_API_KEY")
        )
        self.conversation_history = []

    def chat(self, user_input: str) -> str:
        # 添加用户消息
        self.conversation_history.append({
            "role": "user",
            "content": user_input
        })

        # 调用API
        response = self.client.messages.create(
            model="claude-sonnet-4-6-20250514",
            max_tokens=1024,
            system="你是一个友好、有帮助的AI助手。",
            messages=self.conversation_history
        )

        # 提取回复
        assistant_message = response.content[0].text

        # 添加到历史
        self.conversation_history.append({
            "role": "assistant",
            "content": assistant_message
        })

        return assistant_message

    def clear_history(self):
        self.conversation_history = []

# 使用示例
if __name__ == "__main__":
    assistant = ClaudeAssistant()

    while True:
        user_input = input("你: ")
        if user_input.lower() in ["退出", "quit", "exit"]:
            break

        response = assistant.chat(user_input)
        print(f"Claude: {response}")
