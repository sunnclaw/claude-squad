# 90-Day Technical Plan

## Month 1

Goal: establish the revenue loop.

- Launch a clear landing page with one offer and one CTA
- Implement signup, auth, onboarding, and billing
- Instrument the core funnel in PostHog
- Add Sentry and structured application logging
- Create a minimal admin panel for support and account fixes
- Set up customer feedback capture inside the product

Success checks:

- a user can go from landing page to paid account
- the team can identify funnel drop-off by step
- support can resolve common customer issues without engineering intervention

## Month 2

Goal: improve activation and conversion.

- remove top onboarding friction points
- add lifecycle emails around activation and payment
- improve trial or demo experience
- build the 2 to 3 highest-frequency customer requests only
- add operational automations for lead routing and support triage

Success checks:

- time to first value decreases
- trial to paid conversion improves
- support load per active account decreases

## Month 3

Goal: improve retention and leverage.

- add account health reporting
- add churn and cancellation capture
- strengthen backup, restore, and billing audit paths
- formalize SLIs for uptime and critical workflows
- automate repeated internal tasks that consume founder time

Success checks:

- reasons for churn are visible
- money-impacting events are auditable
- repeated weekly internal work is reduced
