# Pilot Customer Onboarding Checklist

> Use this checklist for every new pilot customer. Complete all items before moving to the next phase.

## Pre-Launch (Before Customer Signs)

- [ ] Customer qualification verified (dental clinic, missed calls problem)
- [ ] Pilot agreement signed
- [ ] Payment method confirmed (first month invoice sent)
- [ ] CRM entry created with all contact details

---

## Phase 1: Day 1 - Account Setup & Phone System Access

### Account Creation
- [ ] Create customer account in dashboard
- [ ] Set up dedicated phone number or forwarding configuration
- [ ] Configure business hours in system
- [ ] Add emergency contact information

### Phone System Setup
- [ ] Configure call forwarding from clinic's main line
- [ ] Test forwarding path (call clinic -> verify redirect)
- [ ] Set up voicemail transcription
- [ ] Configure after-hours routing (if applicable)

### Access Provisioning
- [ ] Send login credentials to primary contact
- [ ] Provide dashboard access URL
- [ ] Schedule Day 2 configuration call

### Communication
- [ ] Send welcome email (use welcome-email-template.md)
- [ ] Confirm receipt of credentials
- [ ] Provide setup instructions document

---

## Phase 2: Day 2-3 - Configuration & Test Calls

### System Configuration
- [ ] Configure notification preferences (SMS, email, app)
- [ ] Set up team member alerts
- [ ] Customize greeting scripts
- [ ] Configure appointment booking integration (if applicable)

### Test Calls
- [ ] Conduct 3-5 test calls during business hours
- [ ] Verify call recording is working
- [ ] Test SMS notifications
- [ ] Test email notifications
- [ ] Confirm caller ID displays correctly

### Quality Verification
- [ ] Review call quality (audio clarity)
- [ ] Verify transcription accuracy
- [ ] Test after-hours routing
- [ ] Document any issues and resolve

### Training
- [ ] Walk through dashboard features
- [ ] Show how to view call logs
- [ ] Explain how to respond to missed calls
- [ ] Answer questions about workflow

---

## Phase 3: Day 7 - First Check-in

### Performance Review
- [ ] Pull call statistics (calls answered, missed, avg response time)
- [ ] Review conversion rate (missed calls -> appointments)
- [ ] Identify any recurring issues
- [ ] Collect customer feedback

### Issue Resolution
- [ ] Address any technical problems
- [ ] Adjust configurations if needed
- [ ] Update notification preferences
- [ ] Document all changes

### Success Metrics
- [ ] Calculate potential revenue recovered
- [ ] Compare to pre-pilot baseline
- [ ] Share initial results with customer

---

## Phase 4: Day 14 - Progress Review

### Performance Analysis
- [ ] Compile 2-week statistics
- [ ] Identify patterns (peak times, common call types)
- [ ] Review customer satisfaction
- [ ] Assess team performance

### Optimization
- [ ] Suggest improvements based on data
- [ ] Implement approved changes
- [ ] Update scripts if needed
- [ ] Add any new features

### Documentation
- [ ] Create monthly performance report draft
- [ ] Document all customizations
- [ ] Prepare renewal discussion points

---

## Phase 5: Day 30 - Results Review & Renewal

### Monthly Report
- [ ] Finalize performance metrics
- [ ] Calculate ROI for customer
- [ ] Document success stories
- [ ] Create before/after comparison

### Renewal Discussion
- [ ] Schedule renewal call with decision maker
- [ ] Present monthly results
- [ ] Discuss pricing for ongoing service
- [ ] Address any concerns
- [ ] Get commitment for next period

### Transition to Ongoing Service
- [ ] Update billing to regular rate (if pilot discount applied)
- [ ] Transfer to standard support queue
- [ ] Remove pilot flags from account
- [ ] Add to customer success roster
