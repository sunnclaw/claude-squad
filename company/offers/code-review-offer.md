# Code Review Offer Pack

Last updated: 2026-03-10
Owner: CTO

## Offer Summary

Ideal customer:
- founder, solo developer, or small engineering team that needs a fast external review

Best fit problems:
- a PR or module needs a second set of eyes
- bugs or architecture risks need to be surfaced quickly
- a team wants prioritized findings instead of generic advice

Not a fit:
- full-time embedded development
- open-ended consulting without scope

Core promise:
- fast, senior-level review with prioritized findings and concrete fixes

Standard turnaround:
- 24 to 72 hours

Revision policy:
- one follow-up clarification round on the report

Price logic:
- base price: CNY 50 per file
- variables that change price: file count, complexity, security sensitivity, requested fixes
- rush fee: +50%

Required customer inputs:
- repo or files to review
- what is changing or what feels risky
- language and framework context

## Packages

### Option A: Fast Start
- Deliverables: review of 1 to 3 files with prioritized findings
- Turnaround: 24 hours
- Price: CNY 50 to CNY 150

### Option B: Standard
- Deliverables: review of up to 10 files with findings and recommended fixes
- Turnaround: 48 hours
- Price: CNY 300 to CNY 500

### Option C: Expanded
- Deliverables: module or project review with findings, risk summary, and remediation guidance
- Turnaround: 72 hours
- Price: CNY 800+
