# Sales Qualification Rubric

> Use this rubric to score every discovery conversation with dental clinics. Score 0-2 on each criterion, then sum for total.

---

## Scoring Criteria

### 1. Missed-Call Pain (0-2)

| Score | Description |
|-------|-------------|
| 0 | Rare or unknown - clinic doesn't perceive missed calls as a problem |
| 1 | Happens sometimes - clinic acknowledges some calls are missed |
| 2 | Frequent and clearly painful - clinic actively frustrated by missed calls |

**Key Questions:**
- "How often do calls go unanswered in a normal week?"
- "What happens when calls are missed?"

### 2. New-Patient Dependence (0-2)

| Score | Description |
|-------|-------------|
| 0 | Mostly existing patients - practice relies on recurring patients |
| 1 | Balanced - mix of new and existing patients |
| 2 | Strong dependence on new patient flow - practice needs new patients to grow |

**Key Questions:**
- "What percentage of your patients are new vs. returning?"
- "How important is new patient acquisition to your growth?"

### 3. Callback Gap (0-2)

| Score | Description |
|-------|-------------|
| 0 | Existing reliable workflow - missed calls are handled consistently |
| 1 | Some inconsistency - callback process is ad-hoc |
| 2 | No dependable follow-up process - calls often fall through cracks |

**Key Questions:**
- "Who owns callbacks when calls are missed?"
- "How quickly do you try to return missed calls?"

### 4. Speed to Value (0-2)

| Score | Description |
|-------|-------------|
| 0 | Hard to measure - clinic doesn't track call outcomes |
| 1 | Some measurable signal - clinic has some tracking capability |
| 2 | Easy to track bookings recovered - clinic can measure results directly |

**Key Questions:**
- "Can you measure how many appointments are booked from calls?"
- "Do you have any call tracking or analytics?"

### 5. Decision Access (0-2)

| Score | Description |
|-------|-------------|
| 0 | No clear owner - decision process unclear |
| 1 | Influencer only - contact can recommend but not approve |
| 2 | Owner or manager engaged - decision-maker is in the conversation |

**Key Questions:**
- "Who would need to approve a pilot?"
- "Are you the decision-maker for new tools?"

### 6. Pilot Urgency (0-2)

| Score | Description |
|-------|-------------|
| 0 | No timeline - no interest in solving soon |
| 1 | Curious but later - interested but not prioritized |
| 2 | Wants to solve in next 30 days - active pain driving urgency |

**Key Questions:**
- "When would you want to start if this makes sense?"
- "What's driving the timeline?"

### 7. Tech Readiness (0-2)

| Score | Description |
|-------|-------------|
| 0 | Resistant - skeptical of new technology |
| 1 | Neutral - open but not tech-forward |
| 2 | Already uses texting or scheduling tools - tech-comfortable |

**Key Questions:**
- "What tools do you currently use for scheduling?"
- "Have you tried texting patients before?"

---

## Total Score Interpretation

| Total Score | Rating | Action |
|-------------|--------|--------|
| 11-14 | Strong Pilot Candidate | Proceed to proposal immediately |
| 7-10 | Nurture Lead | Add to follow-up list, revisit with sharper ROI framing |
| 0-6 | Low-Priority Lead | Thank and disqualify politely |

---

## Disqualifiers (Any = Immediate Disqualification)

- [ ] Cannot describe any missed-call problem
- [ ] No meaningful phone-driven patient acquisition
- [ ] Requires enterprise procurement or DSO approval path
- [ ] Strong policy objection to patient texting or workflow changes
- [ ] Existing solution already handles missed-call follow-up well

## Example Scoring

### Strong Candidate (12/14)
- Missed-call pain: 2 (8-10 missed calls per week, front desk overwhelmed)
- New-patient dependence: 2 (heavily reliant on new patients from Google Ads)
- Callback gap: 2 (no process, calls go to voicemail)
- Speed to value: 2 (can track appointments in practice software)
- Decision access: 2 (owner on the call)
- Pilot urgency: 1 (interested, thinking Q2)
- Tech readiness: 1 (uses basic texting for reminders)

### Nurture Lead (8/14)
- Missed-call pain: 1 (happens occasionally)
- New-patient dependence: 1 (balanced patient base)
- Callback gap: 1 (front desk tries to call back)
- Speed to value: 1 (limited tracking)
- Decision access: 2 (office manager engaged)
- Pilot urgency: 1 (curious but not urgent)
- Tech readiness: 1 (open to tools)
