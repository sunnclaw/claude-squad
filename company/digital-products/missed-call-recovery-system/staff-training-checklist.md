# Staff Training Checklist for Missed Call Recovery

## Front Desk Excellence Program

Use this checklist to train your team on missed call recovery protocols.

---

## Training Overview

**Duration:** 2 hours (can be split into 30-minute sessions)
**Materials Needed:** This checklist, phone scripts, practice phone log
**Trainer:** Office Manager or Lead Front Desk

---

## Section 1: Understanding the Problem (20 minutes)

### Learning Objectives
- [ ] Staff can explain the revenue impact of missed calls
- [ ] Staff can identify peak missed-call times
- [ ] Staff understands the goal: <5 minute response time

### Key Statistics to Share
- Average dental practice misses 5-15 calls per week
- 40% of missed calls are potential new patients
- New patient value: $300-$500
- 20% of missed calls can be recovered with text-back

### Discussion Questions
1. What times of day do we get the most calls?
2. When are we most likely to miss calls?
3. What happens to patients who get voicemail?

---

## Section 2: Detection Protocol (15 minutes)

### Learning Objectives
- [ ] Staff knows how to check for missed calls
- [ ] Staff understands the notification system
- [ ] Staff can access call logs

### Detection Steps
1. [ ] Check phone system for missed calls every 30 minutes
2. [ ] Note caller ID and time of call
3. [ ] Check if caller is existing patient in system
4. [ ] Flag for follow-up if needed

### Practice Exercise
- Simulate 3 missed calls
- Have staff identify and log each one
- Time how quickly they respond

---

## Section 3: Response Protocol (30 minutes)

### Learning Objectives
- [ ] Staff can choose appropriate text template
- [ ] Staff can customize template for situation
- [ ] Staff can send text within 5 minutes of detection

### Template Selection Guide

| Situation | Template |
|-----------|----------|
| Unknown number, business hours | New Patient Focus |
| Known patient | Existing Patient Recognition |
| After hours | After-Hours Response |
| Multiple calls | Multiple Missed Calls |
| Emergency pattern | Emergency Acknowledgment |

### Customization Checklist
- [ ] Practice name inserted
- [ ] Phone number included
- [ ] Patient name used (if available)
- [ ] Appropriate CTA included

### Practice Exercise
- Present 5 scenarios
- Have staff select and customize template
- Review and provide feedback

---

## Section 4: Callback Execution (20 minutes)

### Learning Objectives
- [ ] Staff knows callback priority order
- [ ] Staff has callback script memorized
- [ ] Staff can document outcome

### Callback Priority Order
1. Emergency calls (immediate)
2. Multiple missed calls (within 1 hour)
3. New patient calls (within 2 hours)
4. Existing patient calls (same day)

### Callback Script

**Opening:**
> "Hi, this is [Your Name] from [Practice Name]. I'm returning your call from earlier today."

**If they want to schedule:**
> "I'd be happy to help you schedule an appointment. We have openings on [Day] at [Time] or [Day] at [Time]. Which works better for you?"

**If they have questions:**
> "I can help with that. What would you like to know?"

**Closing:**
> "Is there anything else I can help you with today? Thanks for choosing [Practice Name]!"

### Documentation Requirements
- [ ] Caller name
- [ ] Reason for call
- [ ] Outcome (scheduled, question answered, no answer)
- [ ] Follow-up needed?

---

## Section 5: Quality Assurance (15 minutes)

### Learning Objectives
- [ ] Staff can self-audit performance
- [ ] Staff knows weekly metrics to track
- [ ] Staff understands improvement process

### Daily QA Checklist
- [ ] All missed calls logged
- [ ] All texts sent within 5 minutes
- [ ] All callbacks attempted
- [ ] Outcomes documented

### Weekly Metrics
| Metric | Target | Actual |
|--------|--------|--------|
| Missed calls detected | 100% | |
| Texts sent within 5 min | 95% | |
| Callbacks completed | 90% | |
| Appointments recovered | 2+ | |

### Improvement Process
1. Review weekly metrics every Monday
2. Identify gaps
3. Practice weak areas
4. Re-test

---

## Section 6: Role-Play Practice (20 minutes)

### Scenarios to Practice

**Scenario 1: New Patient Call**
- Trainer pretends to be a new patient
- Staff must use New Patient Focus template
- Staff must handle scheduling request

**Scenario 2: Emergency Call**
- Trainer pretends to be patient with tooth pain
- Staff must use Emergency Acknowledgment template
- Staff must provide emergency line

**Scenario 3: Frustrated Caller**
- Trainer pretends to be annoyed by missed call
- Staff must apologize professionally
- Staff must recover the interaction

**Scenario 4: Multiple Missed Calls**
- Trainer pretends to have called 3 times
- Staff must use Multiple Missed Calls template
- Staff must prioritize callback

---

## Certification Checklist

After training, staff should be able to:

- [ ] Explain the revenue impact of missed calls
- [ ] Demonstrate missed call detection in <2 minutes
- [ ] Select correct template for 5 different scenarios
- [ ] Customize template with practice information
- [ ] Send text response in <5 minutes
- [ ] Execute callback using script
- [ ] Document outcome correctly
- [ ] Complete daily QA checklist

## Ongoing Training

### Weekly (5 minutes)
- Review previous week's metrics
- Highlight successes
- Identify one improvement area

### Monthly (15 minutes)
- Re-test on scenarios
- Update templates if needed
- Share new best practices

### Quarterly (30 minutes)
- Full re-certification
- Update protocols
- Set new targets
