# Weekly Revenue Engineering Review

Run this once per week. Keep it to 45 minutes.

## Agenda

1. Read the scoreboard aloud.
2. Name the current bottleneck.
3. Review all in-flight initiatives.
4. Kill, pause, or continue each one.
5. Set next week's single priority.

## Review Questions

- What moved revenue, activation, retention, or margin last week?
- What did engineering ship that customers actually touched?
- What work consumed time but did not move the scoreboard?
- What manual work is repeating enough to justify automation?
- What did we learn from sales calls, onboarding, or support?
- What should we stop doing now?

## Initiative Review Format

For each active initiative:

- Goal:
- Metric:
- Current evidence:
- Remaining work:
- Risk:
- Decision: continue, narrow, pause, or kill

## End State

At the end of the meeting there must be:

- One company bottleneck.
- One weekly priority.
- A short list of active engineering work with owners.
- A stop-doing list.
