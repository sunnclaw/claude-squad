# Weekly Dashboard Spec

Use one dashboard with the following sections.

## Revenue

- New pipeline created
- New revenue closed
- Expansion revenue
- Average sales cycle length

## Funnel

- Leads to qualified
- Qualified to proposal
- Proposal to close
- Time to first value

## Product

- Activation rate
- Weekly active accounts
- Retained accounts at 4 weeks
- Top 3 friction points

## Engineering

- Cycle time from scoped to production
- Releases per week
- Bugs in paid workflows
- Percent of time on Revenue + Retention

## Review routine

- Monday: choose one outcome based on this dashboard
- Friday: review movement and document lessons
